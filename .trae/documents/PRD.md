## 1. 产品概述

基于 Apple 官网主页风格，为「全能复合运营」黄苑打造高端个人简历展示网站，将个人品牌以 Apple 产品级质感呈现。
- 目标用户：潜在雇主、合作伙伴、招聘方
- 核心价值：以极简、高质感的设计语言，突出黄苑「一人多面、全栈运营」的复合型人才定位

## 2. 核心功能

### 2.1 用户角色
| 角色 | 访问方式 | 核心权限 |
|------|---------|---------|
| 访客 | 直接访问 | 浏览全部内容 |

### 2.2 功能模块
1. **Hero 首屏**：全屏大标题 + 背景视觉，Apple 风格的大字排版
2. **关于我**：个人简介 + 核心数据统计 + 技能标签
3. **工作经历**：时间线式的职业经历展示
4. **实战经验**：项目作品网格展示
5. **技能专长**：分类技能展示
6. **联系我**：联系方式展示

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | Hero 首屏 | 全屏居中大标题、副标题、简介、CTA 按钮、滚动指示器 |
| 首页 | 关于我 | 双栏布局：个人描述 + 统计数字 + 技能标签云 |
| 首页 | 工作经历 | 垂直时间线，展示三段工作经历 |
| 首页 | 实战经验 | 网格卡片展示项目，包含项目图片、描述、技术标签 |
| 首页 | 技能专长 | 三栏分类展示：运营能力、内容制作、综合能力 |
| 首页 | 联系我 | 居中展示联系方式（邮件、电话、微信） |

## 3. 核心流程

用户滚动浏览单页简历网站，通过固定导航栏快速跳转到各个模块。

```mermaid
flowchart TD
    "访客访问网站" --> "浏览 Hero 首屏"
    "浏览 Hero 首屏" --> "向下滚动浏览各个模块"
    "向下滚动浏览各个模块" --> "通过导航栏快速跳转"
    "通过导航栏快速跳转" --> "浏览关于我/经历/项目/技能"
    "浏览关于我/经历/项目/技能" --> "通过联系方式联系"
```

## 4. 用户界面设计

### 4.1 设计风格
- **设计语言**：Apple 极简主义——大量留白、清晰的信息层级、沉浸式视觉
- **主色**：纯黑 (#000) / 深灰 (#1d1d1f) 为底色
- **强调色**：白色 (#f5f5f7) 主文字 + 金色 (#d4a853) 点缀
- **字体**：展示用衬线字体 + 正文用无衬线字体
- **按钮风格**：纯色/描边简约按钮，微圆角
- **布局风格**：大区块式布局，内容居中，大量留白
- **动画风格**：Apple 风格的平滑滚动显现、弹性过渡、视差效果

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页 | 导航栏 | 透明→毛玻璃固定导航，品牌 Logo + 导航链接 |
| 首页 | Hero 首屏 | 全屏背景图 + 暗色遮罩 + 居中文字 + 渐变光晕 |
| 首页 | 关于我 | 双栏不对称布局 + 统计数字 + 标签云 |
| 首页 | 工作经历 | 垂直时间线 + 卡片内容 + 金色圆点指示器 |
| 首页 | 实战经验 | 2×2 网格卡片 + 覆盖背景图 + 悬停效果 |
| 首页 | 技能专长 | 3 列分类卡片 + 进度条动画 |
| 首页 | 联系我 | 居中展示 + 联系方式列表 + 图标 |
| 首页 | 页脚 | 简洁版权声明 |

### 4.3 响应式设计
- 桌面端优先设计
- 平板端 (≤1024px)：双栏变单栏，网格调整为 2 列
- 手机端 (≤768px)：单列布局，导航折叠，字号缩小
- 触摸优化：按钮和链接有足够的触摸区域

### 4.4 动画与交互
- 页面滚动时元素渐入显现（从下往上、从左/右侧滑入）
- 导航栏滚动后变为毛玻璃效果
- 阅读进度条跟随滚动位置
- 时间线卡片悬停微动效
- 项目卡片悬停上移 + 阴影加深
- 统计数字滚动计数动画
- 鼠标跟随金色粒子特效（可选）
